Animación con <animate> (Posición en X)

Este ejemplo utiliza el elemento <animate> para animar la posición en el eje X de un círculo. Puedes ver cómo el círculo se desplaza de la posición inicial a la posición final a lo largo de un intervalo de tiempo especificado.

Animación con <animateMotion> (Trayectoria)

En este ejemplo, utilizamos el elemento <animateMotion> para animar el movimiento de una pelota a lo largo de una trayectoria definida por un elemento <path>. El rectángulo sigue la trayectoria, creando una animación de movimiento.

Animación con <animateTransform> (Rotación)

Este ejemplo utiliza el elemento <animateTransform> para animar la rotación de un triángulo. El triángulo rota 360 grados en sentido horario alrededor de un punto de referencia especificado, creando una animación de rotación.
